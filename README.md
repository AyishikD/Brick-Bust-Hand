# Brick Bust Game with Python, MediaPipe, and OpenCV

A simple Brick Bust game where you control the paddle by swiping left or right using hand gestures. The game is built using Python, MediaPipe for hand tracking, OpenCV for computer vision, and other related libraries.

## Requirements

Make sure you have the following libraries installed:

- Python (>=3.6)
- MediaPipe
- OpenCV
- keyboard

You can install the required libraries using the following command:

```bash
pip install mediapipe opencv-python keyboard
```
## How to Play
Run the game script: 
```bash
python game.py
```
```bash
python hand.py
```
Adjust your hand in front of the camera to start tracking.
Swipe left or right to move the paddle.
Aim to break all the bricks by bouncing the ball off the paddle.

## Getting Started
Clone the repository:
```bash
git clone https://github.com/your-username/Brick-Bust-Hand.git
cd brick-bust-game
```
Install dependencies:
```bash
pip install -r requirements.txt
```
